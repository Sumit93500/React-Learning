## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/react-learning.git
```

Navigate to the project folder:

```bash
cd react-learning
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
